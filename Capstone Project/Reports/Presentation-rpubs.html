Introduction

  • The application is built to predict the next word based on some training data. Data for training was downloaded from the link provided by Coursera.
  • This data includes 899288 blogs, 77259, news articles and 23360148 tweets. For this project only blogs and news articles were used for training.
  • 5-gram Stupid Backoff algorithm was used to predict the next word.
  • Application can be found here
